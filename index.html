# 🏗️ MÓDULO RECLUTAMIENTO GEAT **Automatización completa de reclutamiento de contratistas para GEAT - Construcción y Consultoría** [![Status](https://img.shields.io/badge/status-ready-brightgreen)]() [![Node](https://img.shields.io/badge/node->=16.0-green)]() [![Version](https://img.shields.io/badge/version-1.0.0-blue)]() --- ## 📋 Índice - [Descripción](#descripción) - [Características](#características) - [Instalación](#instalación) - [Configuración](#configuración) - [Uso](#uso) - [API](#api) - [Troubleshooting](#troubleshooting) - [Contribuir](#contribuir) --- ## 📖 Descripción Sistema automático que **sincroniza obras desde OneDrive, publica en Facebook automáticamente, filtra candidatos por Messenger, y guarda resultados en Excel**. **Objetivo:** Reducir tiempo de reclutamiento de 50 horas/mes a 3 horas/mes (94% ahorro). **Stack:** - Backend: Node.js + Express - Frontend: React + Tailwind - Integraciones: OneDrive API, Facebook Graph, Messenger, WhatsApp, Excel --- ## ✨ Características ### ✅ Automáticas - ✓ Lee obras de OneDrive automáticamente - ✓ Publica en Facebook automático (grupo "trabajos en santa cruz-bolivia") - ✓ Bot Messenger responde preguntas automático - ✓ Filtra candidatos (disponibilidad, experiencia, precio) - ✓ Envía planos por WhatsApp automático - ✓ Guarda resultados en Excel automático - ✓ Dashboard en tiempo real ### 🎯 Filtro 1 (Automático en Messenger) ``` Disponibilidad (SÍ/NO) ↓ Aceptación precio (SÍ/NO) ↓ Experiencia en años ↓ Fotos de trabajos previos (2 min) ``` ### 📊 Dashboard - Obras activas - Candidatos por obra - Filtros dinámicos - KPI (conversión, ciclo) - Estados visuales ### 🚀 Integraciones - OneDrive (lectura/escritura) - Facebook (publicar + comentarios) - Messenger (respuestas automáticas) - WhatsApp (envío de planos) - Excel (almacenamiento) --- ## 📦 Instalación ### Requisitos Previos - Node.js >= 16.0 - npm >= 8.0 - Tokens de APIs (ver Configuración) ### Paso 1: Clonar/Descargar ```bash git clone cd geat-reclutamiento ``` ### Paso 2: Instalar dependencias ```bash npm install ``` ### Paso 3: Crear archivo .env ```bash cp .env.template .env # Editar .env con tus tokens nano .env ``` ### Paso 4: Iniciar servidor ```bash npm start ``` ✅ Servidor en: `http://localhost:3000` --- ## ⚙️ Configuración ### 1. ONEDRIVE **Obtener tokens:** 1. Ve a https://portal.azure.com 2. Crea App Registration (nombre: "GEAT-Reclutamiento") 3. Copia: - Client ID - Client Secret - Tenant ID **En .env:** ``` ONEDRIVE_CLIENT_ID=xxx ONEDRIVE_CLIENT_SECRET=xxx ONEDRIVE_TENANT_ID=xxx ``` **Permisos necesarios:** - Files.ReadWrite - Sites.ReadWrite.All --- ### 2. FACEBOOK MESSENGER **Obtener tokens:** 1. Ve a https://developers.facebook.com 2. Crea App (nombre: "GEAT-Reclutamiento") 3. Obtén: - Page Access Token - Page ID - Verify Token (túple lo que quieras) **En .env:** ``` FACEBOOK_PAGE_ID=309806206107796 FACEBOOK_PAGE_TOKEN=xxx FACEBOOK_GRAPH_TOKEN=xxx FACEBOOK_VERIFY_TOKEN=your_secret_token ``` **Configurar Webhook:** 1. Messenger → Webhooks 2. URL: `https://tu-dominio.com/webhook/facebook` 3. Verify Token: mismo del .env 4. Suscribirse a: `messages`, `messaging_postbacks` --- ### 3. FACEBOOK GRAPH (Publicar) ``` FACEBOOK_GRAPH_TOKEN=xxx FACEBOOK_GROUP_ID=637322189611496 ``` **Permisos:** - pages_manage_posts - pages_manage_metadata - pages_read_engagement --- ### 4. WHATSAPP **Opción A: Twilio** 1. Ve a https://www.twilio.com 2. Crea cuenta 3. Obtén API Key + número WhatsApp **Opción B: Meta** 1. Ve a https://business.facebook.com/wa/ 2. Configura WhatsApp Business 3. Obtén token + número **En .env:** ``` WHATSAPP_API_KEY=xxx WHATSAPP_NUMERO=+591XXXXXXXXX ``` --- ## 🚀 Uso ### Iniciar Sistema ```bash npm start ``` **Salida esperada:** ``` ╔═════════════════════════════════════════════════════════════╗ ║ ║ ║ 🚀 SERVIDOR GEAT RECLUTAMIENTO INICIADO ║ ║ ║ ║ 🔗 http://localhost:3000 ║ ║ 📝 Webhooks: /webhook/facebook ║ ║ 📊 APIs: /api/* ║ ║ ║ ║ ✅ Sistema listo para procesar candidatos ║ ║ ║ ╚═════════════════════════════════════════════════════════════╝ ``` ### Flujo Típico ``` 1. Fátima crea OBRA en Excel (OneDrive) ↓ 2. [AUTOMÁTICO] Sistema lee OneDrive ↓ 3. [AUTOMÁTICO] Publica en Facebook ↓ 4. Candidato comenta "Interesado" ↓ 5. [AUTOMÁTICO] Bot Messenger envía Filtro 1 ↓ 6. [AUTOMÁTICO] Si CUMPLE → Envía WhatsApp + planos ↓ 7. [AUTOMÁTICO] Guarda en Excel ↓ 8. Tú ves en WEBAPP → Llamas → Contratas ``` --- ## 📡 API ### GET `/health` Estado del servidor ```bash curl http://localhost:3000/health ``` ### GET `/api/estado` Estado actual del sistema ```bash curl http://localhost:3000/api/estado ``` **Response:** ```json { "estado": { "activo": true, "obrasActivas": 2, "candidatosPendientes": 5, "ultimaSincronizacion": "2026-04-01T14:00:00Z" } } ``` ### GET `/api/obras` Obtener todas las obras ```bash curl http://localhost:3000/api/obras ``` **Response:** ```json { "obras": { "OBRA_037": { ... }, "OBRA_038": { ... } }, "total": 2 } ``` ### GET `/api/candidatos/:obraId` Obtener candidatos de una obra ```bash curl http://localhost:3000/api/candidatos/037 ``` **Response:** ```json { "obraId": "037", "candidatos": [ { "nombre": "Juan Pérez", "telefono": "76123456", "zona": "La Paz", "estado": "filtro1_ok" } ], "total": 5, "filtro1OK": 3 } ``` ### POST `/api/publicar-obra` Publicar obra en Facebook ```bash curl -X POST http://localhost:3000/api/publicar-obra \ -H "Content-Type: application/json" \ -d '{ "obraId": "037", "nombre": "Vivienda Quisbert", "zona": "La Paz", "presupuesto": 78, "m2": 154.35, "duracion": "150 días", "fechaInicio": "31/03/2026" }' ``` ### POST `/api/enviar-whatsapp` Enviar planos por WhatsApp ```bash curl -X POST http://localhost:3000/api/enviar-whatsapp \ -H "Content-Type: application/json" \ -d '{ "numero": "591xxxxxxxxx", "obraId": "037", "nombre": "Vivienda Quisbert", "presupuesto": 78, "enlacePlanos": "https://drive.google.com/..." }' ``` ### GET `/api/metricas` Obtener KPI del sistema ```bash curl http://localhost:3000/api/metricas ``` **Response:** ```json { "obras": 2, "totalCandidatos": 12, "filtro1OK": 8, "contratados": 1, "tasaConversion": "66.7%", "tiempoPromedioCierre": "5 días" } ``` --- ## 🐛 Troubleshooting ### ❌ "ONEDRIVE_TOKEN no definido" **Solución:** Verifica archivo `.env` existe y tiene `ONEDRIVE_CLIENT_ID` ### ❌ "Error conectando OneDrive" **Solución:** Valida tokens en https://portal.azure.com ### ❌ "Facebook webhook no responde" **Solución:** Verifica FACEBOOK_VERIFY_TOKEN coincida en settings ### ❌ "WhatsApp no envía mensajes" **Solución:** Valida número está verificado en Twilio/Meta ### ❌ "Excel no se sincroniza" **Solución:** Verifica permisos de Azure (Files.ReadWrite) --- ##
